# Shared Laboratory Booking Calendar
This is the lab booking calendar of the Mixed Reality Lab at the Frankfurt University of Applied Sciences. The booking calendar can block multiple seats, rooms and spaces for multiple user studies. Checkour the Group-Backend to organize your rooms and groups.
